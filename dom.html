Document Object Model

The DOM is a programming interface that repesents the structure of a html document in a tree like structure (a javascript object)

Each node in the tree represents a part of the document secu as elements, attributes and text.

The DOM provides a way for Javascript to interact with HTML elements and manipulate their content, structure and style dynamically. This means you can use javascript to create, modify or delete elements on a web page, respond to user actions and update the content without needing to reload the entire page.

The first step to manipulating an element is selecting the element is selecting the element using methods like getElementById(), querySelector(), getElementByClassName()
